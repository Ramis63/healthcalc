HealthCalc is a fast, privacy-friendly command-line tool for calculating your BMI (Body Mass Index) and getting instant health tips.
Built for speed and simplicity, it runs offline and collects no data.

Download for Windows Download for Mac Download for Linux

How to Use

  • Download and save the file for your system above.
  • Open your terminal or command prompt.
  • Run the HealthCalc tool:
    Windows: healthcalc-windows.exe
    Mac/Linux: ./healthcalc-macos or ./healthcalc-linux
  • Follow the on-screen instructions.

Features

  • Instant BMI calculation
  • Health category detection (underweight, normal, overweight, obese)
  • Personalized health tips
  • Fast and 100% offline
